开专栏的原因
1.检测自己的自学能力；
2.测试自己是否有文笔，看这几年的阅历是否增加，是否变得有趣；
3.检测自己是否还是和做RD；
4.检测自己是否能坚持今年的目标，27年来自己定的一个硬性目标。

为什么选择Ionic
在3年前，被逼之下操作双端，从Android native开发强制进入js领域，hybrid的学习，从简单的js到angular 1，从angular 1 到ionic1，从web app到Android加载web app，从Android到ios加载ionic是有点情怀；

为什么突然用ionic写项目
自己的旅游咨询想法蠢蠢欲动了2年，就跟自己决定前往澳洲读书一样都停留在嘴上，光打嘴炮就打了4-5年，不允许自己将在30岁之前光打嘴炮，而是要坚定的往前迈一步，即使很艰难。所以在自己在2月立即决定自己去澳洲读书，立马就参加了雅思6.5分班，立马找了中介，立马准备了材料，立马投了学校，虽然遇到了新冠病毒的影响，雅思考试推迟，也同样给我备考增加了时间，顺利将学期推迟到明年2月，希望带着这份专栏的热情和对留学更深蒂固的执着，雅思顺利上岸。扯远了，希望自己的创业梦想同样实现，虽然不能太贪心，但是这个想着慢慢来吧，不急不急，同样感谢现在公司的领导和身边的朋友支持我，尤其是家里人的支持，让我能这样操作一波，这就是我写ionic项目的原因，旨在用ionic开发一款旅游咨询软件。

📈从何下手？
刚刚看了下这几年的版本变更是在太快，是在太多，从cordova到了现在的capacitor（电容器）变化真的多，所以老步骤开始！

🔧Install Schedule
npm install -g @ionic/cli
ionic start myApp blank –type=react

💥前方高等，提前声明
这里用的react，原因是目前公司有个项目用RN开发所以先熟悉react语法及使用，同时挑战从头开始学js与ts，因为刚刚瞄了一眼项目居然是tsx的，顿时有点瑟瑟发抖。同时，这里有可能会变成一个这个App所用到所有知识的细化专栏，不一定限于ionic，以ionic为主。
